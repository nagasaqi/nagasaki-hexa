Nagasaqi-Hexa is a very small sized project to develop MIDI synthesizer for fun.

It tweaks existing GUI classes and Mouse-listeners to get user input and uses MIDI Synthesizer to generate sounds. Vision is to make maximum out of minimum resources. I future using Multi-Pointer Aware backend, We hope to create a scalable and cheap synthesizer for all.
